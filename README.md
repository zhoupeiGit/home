# 花儿与糖果

```
	所以我们要用正确的方式去教育祖国的花朵...

	什么又是正确的方式呢? 谁又不曾是祖国的小花朵。
	每个人教育方式行为都不一样,简单的关心,严肃的批评,或者处分,但是个人觉得离不开感觉，好感,存在感,内疚感,成就感,就如同陶行知的教育方式，首先,一开始就给犯错误的小朋友一颗糖,小朋友面对受处罚的小心心也就放下来了,至少感觉不坏,给第二块糖时,咦,居然不惩罚我反而还奖励我,这个时候好有存在感,我可是刚刚做错事情了呀,还间接的让小朋友懂得尊重原则，第三颗糖,小朋友直接被感动了,留着眼泪主动承认了错误,感觉到自己特别内疚,还让陶校长打两下,直至第四颗糖给小朋友,表扬小朋友,让小朋友觉得这样才是正确的。所有,这几颗糖果留下了无法忘记的感动,应该很甜叭。
	祖国的小花朵,曾今是,现在......
	嘿嘿,紧记园丁栽培的大花朵,以后要如同春雨般的方式去教育小花朵们啦。
```
